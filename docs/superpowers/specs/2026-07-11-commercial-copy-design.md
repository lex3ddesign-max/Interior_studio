# Commercial Copy Design

## Goal

Усилить текст AVENOR так, чтобы сайт звучал как спокойная экспертная студия 3D-визуализации для интерьерных, архитектурных и коммерческих проектов.

## Approved tone

- Quiet luxury / экспертный.
- Без крика, без «вау-эффекта», без дешёвых обещаний.
- Коммерческая польза должна быть понятной: меньше сомнений, быстрее согласование, сильнее презентация, меньше переделок.
- Тексты должны оставаться короткими и хорошо ложиться в текущую сетку.

## Scope

- Главная: hero, позиционирование, кейсы, услуги, процесс, стоимость, доверие, контакты.
- Услуги: описания направлений, списки включённых работ, FAQ, общий процесс.
- Стоимость: пакеты, пояснения, факторы цены, CTA.
- Кейсы: подзаголовки, описание задачи, highlights, техническая документация.
- О нас: студийный текст и принципы.
- SEO descriptions: только там, где текст явно общий или слабый.

## Non-goals

- Не менять визуальную структуру страниц.
- Не менять цены и сроки из таблицы.
- Не добавлять новые интеграции формы.
- Не заменять изображения.

## Success criteria

- На сайте не остаётся сухих формулировок уровня «показали / сделали / подготовили» без пользы для клиента.
- FAQ отвечает на практические вопросы и звучит уверенно.
- Тексты не ломают верстку и не становятся длиннее текущих блоков критично.
- `npm test -- --run`, `npm run lint`, `npm run build` проходят.
