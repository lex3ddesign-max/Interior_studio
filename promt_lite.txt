Ты — senior frontend developer и creative technologist. Нужно разработать премиальный frontend MVP сайта для visual production-студии AVENOR.

Проект: AVENOR — студия 3D-визуализации интерьеров, экстерьеров и коммерческих пространств.

Главная задача сайта:
создать эффектный промо-сайт в стиле dark cinematic quiet luxury, который продаёт услугу через атмосферу, сильные рендеры, cinematic-анимации и красивую подачу кейсов.

Не делай обычный шаблонный лендинг. Сайт должен ощущаться как премиальный digital-кейс visual production-студии.

====================================
1. СТЕК
====================================

Используй:

- TypeScript
- Tailwind CSS
- GSAP

- Lenis

9.1 HeroRevealSequence
------------------------------------

Hero — самая важная часть сайта.

Сделай 3 сцены:

1. интерьер;
2. экстерьер;
3. коммерческое пространство.

Сцены должны сменяться не как обычный slider, а через reveal/mask/clip-path переход.

Сценарий:

- первый рендер открывается через прямоугольную архитектурную маску;
- при скролле сцены сменяются;
- текст остаётся стабильным;
- изображения получают лёгкий parallax/zoom;
- после третьей сцены пользователь попадает в основной контент.

Используй GSAP + ScrollTrigger.

На desktop можно использовать pinned section.
На mobile упростить до crossfade/reveal без тяжёлого pinned-scroll.

Визуализируем пространства,
в которые хочется войти
